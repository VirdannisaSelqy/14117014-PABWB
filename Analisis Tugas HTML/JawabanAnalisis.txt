3.	Jelaskan fungsi tag HTML pada Table: thead, tbody, tr, th serta td!
        Jawab :
        1. HTML <thead> element digunakan untuk mengelompokkan konten yang terletak di kepala (atas) sebuah tabel.
	2. Di dalam element <thead> tersebut, harus ditulis element <tr> yang menunjukkan table row atau baris sebuah table. 
	3. HTML <tbody> element digunakan untuk mengelompokkan konten yang terletak pada bagian tubuh table itu sendiri.
 	4. <th> : kepanjangan dari table heading, digunakan untuk membuat judul sebuah table seperti No, dan nama. 
           biasanya di tanda dengan hurufnya menjadi bold seperti contoh table di atas.
	5. <td> : kepanjangan dari table data, digunakan untuk membuat isi dari th atau baris atau kalau di MS.excel seperti cell.
	6. <tr> : kepanjangan dari table row, digunakan untuk mendefiniskan baris pada tabel
